
# **T07: Accés remot. Serveis d’assistència remota (Marti i Marc)**

**Fase 1: Anàlisi Comparativa i Selecció de la Solució:**

Taula comparativa (TeamViewer, AnyDesk, Google Remote Desktop, Zoho Assist)

| Eina | Facilitat per al client | Disponibilitat OS | Model de preu | Notes clau |
| ----- | ----- | ----- | ----- | ----- |
| TeamViewer | Mòdul QuickSupport portable; ID i contrasenya visibles | Windows, macOS, Linux, iOS, Android | Llicència comercial de pagament; limitacions en ús gratuït | Molt complet (transferència d’arxius, reinici) però més car |
| AnyDesk | Portable, fitxer lleuger; codi d’adreça fàcil | Windows, macOS, Linux, FreeBSD, Raspberry Pi, iOS, Android | Ús comercial de pagament; gratuït limitat | Molt ràpid i lleuger; transferència integrada |
| Google Remote Desktop | Instal·lació via Chrome, requereix extensió/servei | Windows, macOS, Linux, ChromeOS | Gratuït; sense llicència comercial | Simplicitat alta però menys funcions avançades |
| Zoho Assist | Enllaç de sessió per a client; agent sota demanda | Windows, macOS, Linux | Llicència per tècnic; opcions gratuïtes limitades | Orientat a helpdesk, cost moderat, bones funcions empresarials |

Recomanació

teamviewer

Opció recomanada:

equilibri molt bo entre facilitat per al client (portable, sense instal·lació), compatibilitat extensa (Windows/macOS/Linux), rendiment i cost. Ideal per connexions ràpides en entorns restrictius sense canviar configuracions de xarxa del client. TeamViewer és molt complet però acostuma a ser més car; Google Remote Desktop és gratuït però limitat en funcions; Zoho Assist és sòlid per helpdesk però implica llicències per tècnic i flux web
